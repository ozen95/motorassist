# MotorAssist

**MotorAssist** is an open-source accessibility tool that helps people with motor disabilities configure their computer software quickly and easily.

## Purpose

Many people with motor impairments (tremors, limited mobility, one-handed use, etc.) struggle to find and apply the right accessibility settings on their computer. MotorAssist guides users through a simple questionnaire and automatically applies the appropriate system settings on **Windows** and **macOS**.

## Features

- Interactive questionnaire to identify user needs
- - Automatic configuration of:
  -   - Sticky Keys (for one-handed typing)
      -   - Slow Keys (for users with tremors)
          -   - Mouse speed reduction
              -   - Large mouse pointer
                  - - Software recommendations tailored to the user's profile
                    - - 100% local — no data collection, no internet required
                      - - Cross-platform: Windows & macOS
                       
                        - ## Getting Started
                       
                        - ### Requirements
                        - - Python 3.8 or higher
                         
                          - ### Installation
                         
                          - ```bash
                            git clone https://github.com/ozen95/motorassist.git
                            cd motorassist
                            python main.py
                            ```

                            > **Windows users:** Run as Administrator for full settings access.
                            > > **macOS users:** Grant Terminal accessibility permissions in System Settings > Privacy & Security.
